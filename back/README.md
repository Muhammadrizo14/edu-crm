## Edu-crm Backend


## Installation

```bash
$ npm install
```

## Running the app

```bash
# in .env file write way to your db

# and in /prisma/schema.prisma change provider  to your db name, <a href="https://www.prisma.io/docs/orm/overview/databases/mysql" target="_blank">View more here!</a>

$ npm i

$ npx prisma init 

$ npx prisma migrate dev --name init

$ npm run start 

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
